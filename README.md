# Covid-19-Analysis
COVID-19 EDA project analyzing global pandemic trends using Python (Pandas, Matplotlib). Visualizes confirmed, recovered, and death cases across countries to uncover patterns, growth trends, and insights for better understanding of the pandemic’s impact.
# COVID-19 Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on global COVID-19 datasets to understand the spread and impact of the pandemic.

## 🚀 Overview
The analysis focuses on confirmed, recovered, and death cases across different countries and time periods. Using Python libraries like Pandas and Matplotlib, the project uncovers trends, growth patterns, and key insights through data visualization.

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Google Colab

## 📊 Key Insights
- Country-wise comparison of cases  
- Growth trends over time  
- Analysis of recovery and mortality rates  

## 🎯 Objective
To extract meaningful insights from COVID-19 data and present them visually for better understanding and decision-making.
